Environment and interpretation

These numbers are repository evidence, not a universal hardware-independent promise.

Comparable

Same machine, same runtime

Every before/after or WPF/Cerneala pairing was captured under the same exposed hardware, runtime, build configuration, and benchmark job.

Structural truth

Tests guard the invariants

Zero-allocation idle paths, exact-once dispatch, bounded backlog, tree traversal counts, and detach cleanup are asserted separately from timing noise.

Boundary

Small system, focused contract

Relay wins a deliberately narrow queue-and-frame workload. WPF Dispatcher still provides priorities, timers, message integration, nested loops, and decades of compatibility semantics.

Noise policy

Ratios are ranges

Short fixed-size iterations produce broad confidence intervals. Repeat on equivalent hardware before calling a small timing delta a regression.
